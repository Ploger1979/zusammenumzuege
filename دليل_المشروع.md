# دليل المشروع (Zusammen Umzüge Manager)

هذا المستند يشرح كيفية عمل لوحة التحكم الخاصة بموقع "Zusammen Umzüge" وكيفية إدارة الطلبات والفواتير والإداريين.

## 1. الدخول إلى لوحة التحكم (Login)

- **الرابط:** `zusammenumzuege.de/admin`
- عند الدخول لهذا الرابط، سيتم توجيهك إلى صفحة تسجيل الدخول.
- **البيانات المطلوبة:** البريد الإلكتروني وكلمة المرور.
- **بعد التسجيل:** يتم تحويلك تلقائياً إلى صفحة **الطلبات (Requests Dashboard)**.

## 2. لوحة التحكم - الطلبات (Request Dashboard)

هذه هي الصفحة الرئيسية للأدمن، وفيها تجد:
- **قائمة الطلبات:** جميع طلبات العملاء الجديدة تظهر هنا مرتبة حسب الأحدث.
- **تفاصيل الطلب:** اسم العميل، بيانات الاتصال، وتاريخ الطلب.
- **الإجراءات:**
    - **زر "إنشاء فاتورة" (Rechnung erstellen):** عند الضغط عليه، يتم تحويلك إلى صفحة **إنشاء الفاتورة** مع تعبئة بيانات العميل (الاسم والعنوان) تلقائياً لتسهيل العمل.
- **إدارة المشرفين:** يوجد زر في الأعلى "Admins" ينقلك لصفحة إدارة المستخدمين.
- **تسجيل الخروج:** يوجد زر "Logout / Abmelden" باللون الأحمر لتسجيل الخروج بأمان.

## 3. صفحة إدارة المشرفين (Admin Management)

- **الوصول:** من خلال زر "Admins" في صفحة الطلبات.
- **الوظيفة:**
    - إضافة مشرفين (Admin) جدد للنظام.
    - حذف مشرفين حاليين.
    - رؤية قائمة بجميع المشرفين المسجلين.

## 4. صفحة إنشاء الفاتورة (Invoice Generator)

هذه الصفحة هي الأداة الرئيسية لعمل الفواتير وتحويلها لملفات PDF.
- **الوصول:** عبر زر "Rechnung erstellen" من أي طلب في لوحة التحكم.
- **المميزات:**
    - **تعبئة تلقائية:** بيانات العميل من الطلب يتم نقلها تلقائياً للفاتورة.
    - **إضافة بنود:** يمكنك إضافة خدمات (نقل، تغليف، كراتين...) مع تحديد السعر والكمية.
    - **حساب تلقائي:** النظام يحسب المجموع والضريبة تلقائياً.
    - **كراتين:** خانة خاصة لحساب عدد الكراتين.
    - **الطباعة (PDF):** زر "طباعة / حفظ PDF" لإنشاء ملف الفاتورة النهائي وإرساله للعميل.
    - **زر الرجوع:** أضفنا زر "Dashboard" للعودة السريعة إلى قائمة الطلبات دون الحاجه لاستخدام المتصفح.

## 5. ملاحظات هامة

- **الأمان:** النظام محمي بكلمة مرور، ويتم تسجيل خروجك تلقائياً بعد فترة لضمان الأمان.
- **اللغات:** لوحة التحكم تدعم الألمانية والإنجليزية والعربية (حسب لغة المتصفح أو اختيارك).
- **التصميم:** يدعم الوضع الليلي (Dark Mode) والوضع النهاري (Light Mode) بشكل تلقائي ومريح للعين.
